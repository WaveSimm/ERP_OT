### 1.2 미니 타임라인 렌더링 방식 결정

**선택: SVG 인라인 바 렌더링** (서버에서 이벤트 데이터 → 클라이언트에서 SVG 렌더링)

| 방식 | 장점 | 단점 | 결정 |
|------|------|------|------|
| SVG 바 (클라이언트 렌더링) | 픽셀 정밀도, 상호작용 | JS 렌더링 비용 | ✅ 선택 |
| 이벤트 목록 텍스트 | 단순, 빠름 | 시각성 부족 | - |
| Canvas | 고성능 | 접근성 저하 | - |
| 서버사이드 SVG | SEO | 상호작용 불가 | - |

**SVG 타임라인 설계**:
- 가로: 15일 (오늘 -7 ~ +7일), 1일 = 일정 너비 단위
- 오늘 = 중앙 세로선 강조 (`stroke: orange, dash`)
- 세그먼트 바: 높이 10px, `rx="3"` 둥근 모서리
- 마일스톤: 다이아몬드 아이콘 (`◆`)
- 지연된 세그먼트: 빨강 `#EF4444`, 정상: 초록 `#22C55E`, 크리티컬: 주황 `#F97316`
- hover 툴팁: 세그먼트명, 담당자, 진행률

## 5. 이슈 감지 엔진 전체 설계

### 5.1 감지 규칙 전체 목록

```typescript
// application/dashboard/issue-detector.service.ts

enum IssueSeverity { CRITICAL = 1, WARNING = 2, INFO = 3 }

interface IssueRule {
  id: string;
  severity: IssueSeverity;
  category: IssueCategory;
  label: string;
  check: (ctx: IssueCheckContext) => IssueCheckResult | null;
}

interface IssueCheckContext {
  project: ProjectWithTasks;
  thresholds: IssueThresholdConfig;
  today: Date;
  windowStart: Date;   // today - 7
  windowEnd: Date;     // today + 7
}

const ISSUE_RULES: IssueRule[] = [
  // ───── CRITICAL ─────────────────────────────────────────────
  {
    id: 'CRITICAL_PATH_DELAYED',
    severity: IssueSeverity.CRITICAL,
    category: 'SCHEDULE_DELAY',
    label: '크리티컬 패스 지연',
    check: ({ project, thresholds, today }) => {
      const delayedCritical = project.tasks
        .filter(t => t.isCritical && t.effectiveEndDate < today && t.status !== 'DONE');
      if (!delayedCritical.length) return null;
      return {
        tasks: delayedCritical,
        maxDelay: Math.max(...delayedCritical.map(t => dateDiff(t.effectiveEndDate, today))),
      };
    },
  },
  {
    id: 'BUDGET_CRITICAL',
    severity: IssueSeverity.CRITICAL,
    category: 'BUDGET_OVERRUN',
    label: `예산 ${thresholds.criticalBudgetPercent}% 초과`,
    check: ({ project, thresholds }) => {
      if (!project.plannedBudget) return null;
      const pct = (project.actualBudget / project.plannedBudget) * 100;
      if (pct < thresholds.criticalBudgetPercent) return null;
      return { overrunPercent: pct };
    },
  },
  {
    id: 'RESOURCE_CRITICAL',
    severity: IssueSeverity.CRITICAL,
    category: 'RESOURCE_OVERLOAD',
    label: `자원 ${thresholds.criticalResourcePercent}% 과부하`,
    check: ({ project, thresholds, today, windowStart, windowEnd }) => {
      const overloaded = getOverloadedResources(
        project, windowStart, windowEnd, thresholds.criticalResourcePercent
      );
      return overloaded.length ? { resources: overloaded } : null;
    },
  },

  // ───── WARNING ──────────────────────────────────────────────
  {
    id: 'NON_CRITICAL_DELAYED',
    severity: IssueSeverity.WARNING,
    category: 'SCHEDULE_DELAY',
    label: '태스크 지연',
    check: ({ project, thresholds, today }) => {
      const delayed = project.tasks.filter(t =>
        !t.isCritical &&
        t.effectiveEndDate < today &&
        t.status !== 'DONE' &&
        dateDiff(t.effectiveEndDate, today) >= thresholds.warningDelayDays
      );
      return delayed.length ? { tasks: delayed } : null;
    },
  },
  {
    id: 'BUDGET_WARNING',
    severity: IssueSeverity.WARNING,
    category: 'BUDGET_OVERRUN',
    label: `예산 ${thresholds.warningBudgetPercent}% 초과`,
    check: ({ project, thresholds }) => {
      if (!project.plannedBudget) return null;
      const pct = (project.actualBudget / project.plannedBudget) * 100;
      if (pct < thresholds.warningBudgetPercent || pct >= thresholds.criticalBudgetPercent) return null;
      return { overrunPercent: pct };
    },
  },
  {
    id: 'RESOURCE_WARNING',
    severity: IssueSeverity.WARNING,
    category: 'RESOURCE_OVERLOAD',
    label: `자원 ${thresholds.warningResourcePercent}% 과부하`,
    check: ({ project, thresholds, windowStart, windowEnd }) => {
      const overloaded = getOverloadedResources(
        project, windowStart, windowEnd, thresholds.warningResourcePercent
      );
      const filtered = overloaded.filter(r =>
        r.allocationPercent < thresholds.criticalResourcePercent
      );
      return filtered.length ? { resources: filtered } : null;
    },
  },
  {
    id: 'PROGRESS_STALE',
    severity: IssueSeverity.WARNING,
    category: 'PROGRESS_STALE',
    label: `${thresholds.warningStaleProgressDays}일 이상 업데이트 없음`,
    check: ({ project, thresholds, today }) => {
      const stale = project.tasks.filter(t =>
        t.status === 'IN_PROGRESS' &&
        dateDiff(t.updatedAt, today) >= thresholds.warningStaleProgressDays
      );
      return stale.length ? { tasks: stale } : null;
    },
  },

  // ───── INFO ─────────────────────────────────────────────────
  {
    id: 'MILESTONE_DUE',
    severity: IssueSeverity.INFO,
    category: 'MILESTONE_DUE',
    label: `마일스톤 ${thresholds.infoMilestoneDueDays}일 이내`,
    check: ({ project, thresholds, today }) => {
      const dueSoon = project.milestones.filter(m => {
        const daysUntil = dateDiff(today, m.dueDate);
        return daysUntil >= 0 && daysUntil <= thresholds.infoMilestoneDueDays;
      });
      return dueSoon.length ? { milestones: dueSoon } : null;
    },
  },
  {
    id: 'SEGMENT_ENDING_THIS_WEEK',
    severity: IssueSeverity.INFO,
    category: 'SCHEDULE_DELAY',
    label: '이번 주 태스크 완료 예정',
    check: ({ project, windowStart, windowEnd }) => {
      const endingThisWeek = project.tasks.flatMap(t =>
        t.segments.filter(s =>
          s.endDate >= windowStart && s.endDate <= windowEnd && s.progressPercent < 100
        )
      );
      return endingThisWeek.length ? { segments: endingThisWeek } : null;
    },
  },
];
```

### 5.3 이슈 감지 실행 흐름

```typescript
class IssueDetectorService {
  async detectForProject(projectId: string): Promise<DashboardIssue[]> {
    // 1. 프로젝트 전체 데이터 로드 (태스크 + 세그먼트 + 자원 + Baseline)
    const project = await this.projectRepo.findWithFullContext(projectId);
    const thresholds = await this.getThresholds(); // Redis 캐시

    const today = startOfDay(new Date());
    const ctx: IssueCheckContext = {
      project,
      thresholds,
      today,
      windowStart: subDays(today, 7),
      windowEnd: addDays(today, 7),
    };

    // 2. 모든 규칙 실행
    const issues: DashboardIssue[] = [];
    for (const rule of ISSUE_RULES) {
      const result = rule.check(ctx);
      if (result) {
        issues.push(buildIssue(rule, result, project));
      }
    }

    // 3. severity 순 정렬 (Critical 먼저)
    return issues.sort((a, b) => a.severity.localeCompare(b.severity));
  }
}
```

## 6. 대시보드 집계 서비스 설계

```typescript
// application/dashboard/dashboard.service.ts

class DashboardService {
  async getDashboard(
    userId: string,
    options: { groupBy: GroupType; date: Date; issueFilter: string; groupId?: string }
  ): Promise<DashboardResponse> {

    // 1. 전체 요약 (캐시 우선)
    const globalSummary = await this.cache.getOrSet(
      'dashboard:global:summary',
      300,
      () => this.computeGlobalSummary()
    );

    // 2. 그룹 계층 조회 (DB, 자주 변경되지 않으므로 TTL 600s)
    const groups = await this.groupService.getGroupHierarchy(options.groupBy);

    // 3. 그룹별 롤업 (캐시 우선, 병렬)
    const rollupPromises = groups.map(g =>
      this.cache.getOrSet(
        `dashboard:group:${g.id}:rollup`,
        300,
        () => this.computeGroupRollup(g.id)
      )
    );
    await Promise.all(rollupPromises);

    // 4. 프로젝트별 요약 + 이슈 (캐시 우선, 병렬 배치)
    const allProjectIds = extractAllProjectIds(groups);
    const projectSummaries = await this.batchGetProjectSummaries(
      allProjectIds, options.date
    );

    // 5. 이슈 필터 적용
    const filtered = applyIssueFilter(projectSummaries, options.issueFilter);

    // 6. 사용자 설정 (즐겨찾기, 펼침 상태)
    const config = await this.getUserConfig(userId);

    return buildDashboardResponse(groups, filtered, globalSummary, config);
  }

  // 캐시 MISS 시 단건 프로젝트 요약 계산
  async computeProjectSummary(projectId: string, date: Date): Promise<ProjectRowSummary> {
    const [project, issues, timeline] = await Promise.all([
      this.projectRepo.findWithProgress(projectId),
      this.issueDetector.detectForProject(projectId),
      this.timelineService.getWeeklyEvents(projectId, date),
    ]);

    return {
      id: project.id,
      name: project.name,
      status: project.status,
      ragStatus: determineRAGStatus(issues),
      overallProgress: project.overallProgress,
      projectedEndDate: project.effectiveEndDate?.toISOString(),
      plannedBudget: project.plannedBudget,
      actualBudget: project.actualBudget,
      budgetUsagePercent: calcBudgetPercent(project),
      isCriticalPathDelayed: issues.some(i => i.id === 'CRITICAL_PATH_DELAYED'),
      issueCount: countIssues(issues),
      weeklyTimeline: timeline,
      isPinned: false, // 사용자 설정에서 주입
      lastUpdatedAt: project.updatedAt.toISOString(),
    };
  }

  // 배치 처리 (병렬 50개씩)
  async batchGetProjectSummaries(
    projectIds: string[], date: Date
  ): Promise<ProjectRowSummary[]> {
    const BATCH_SIZE = 50;
    const results: ProjectRowSummary[] = [];

    for (let i = 0; i < projectIds.length; i += BATCH_SIZE) {
      const batch = projectIds.slice(i, i + BATCH_SIZE);
      const batchResults = await Promise.all(
        batch.map(id =>
          this.cache.getOrSet(
            `dashboard:project:${id}:summary`,
            300,
            () => this.computeProjectSummary(id, date)
          )
        )
      );
      results.push(...batchResults);
    }
    return results;
  }
}
```

```typescript
class DashboardRefreshJob {
  // Cron: 매 5분 실행
  @Cron('*/5 * * * *')
  async run(): Promise<void> {
    // 중복 실행 방지 (분산 환경 대비)
    const locked = await this.redis.set(
      'dashboard:refresh:lock', 'locked',
      'NX', 'EX', 60  // 60초 만료
    );
    if (!locked) return;

    try {
      const activeProjects = await this.projectRepo.findActiveProjectIds();
      const BATCH = 20;

      for (let i = 0; i < activeProjects.length; i += BATCH) {
        const batch = activeProjects.slice(i, i + BATCH);
        await Promise.all(batch.map(async (projectId) => {
          // 기존 캐시 무효화 후 재계산
          await this.redis.del(`dashboard:project:${projectId}:summary`);
          const summary = await this.dashboardService.computeProjectSummary(
            projectId, new Date()
          );

          // 새 Critical 이슈 감지 시 WebSocket 푸시
          const prevSummary = await this.getPreviousSummary(projectId);
          if (hasNewCriticalIssues(prevSummary, summary)) {
            await this.wsGateway.broadcast(`project:${projectId}`, {
              event: 'dashboard:new_critical_issue',
              data: summary,
            });
            // 대시보드 구독자에게도 알림
            await this.wsGateway.broadcast('dashboard:all', {
              event: 'dashboard:issue_alert',
              data: { projectId, summary },
            });
          }
        }));
      }

      // 전체 집계 갱신
      await this.redis.del('dashboard:global:summary');
    } finally {
      await this.redis.del('dashboard:refresh:lock');
    }
  }
}
```

## 9. 미니 타임라인 데이터 집계

```typescript
// application/dashboard/timeline.service.ts

class TimelineService {
  async getWeeklyEvents(projectId: string, centerDate: Date): Promise<TimelineEvent[]> {
    const windowStart = subDays(startOfDay(centerDate), 7);
    const windowEnd = addDays(startOfDay(centerDate), 7);

    // ±7일 범위와 겹치는 세그먼트 조회
    const segments = await this.prisma.taskSegment.findMany({
      where: {
        task: { projectId },
        startDate: { lte: windowEnd },
        endDate:   { gte: windowStart },
      },
      include: {
        task: { select: { id: true, name: true, isCritical: true } },
        assignments: {
          include: {
            resource: { select: { id: true, name: true } },
          },
        },
      },
      orderBy: { startDate: 'asc' },
    });

    // 마일스톤 조회 (±7일 내 태스크 완료일 기준)
    const milestones = await this.prisma.milestone.findMany({
      where: { projectId },
    });

    const events: TimelineEvent[] = [];

    for (const seg of segments) {
      const isDelayed = seg.endDate < centerDate && seg.progressPercent < 100;
      const delayDays = isDelayed ? dateDiff(seg.endDate, centerDate) : 0;

      // 세그먼트 시작 이벤트
      if (seg.startDate >= windowStart && seg.startDate <= windowEnd) {
        events.push({
          date: seg.startDate.toISOString().split('T')[0],
          type: 'SEGMENT_START',
          taskId: seg.task.id,
          taskName: seg.task.name,
          segmentId: seg.id,
          segmentName: seg.name,
          progressPercent: seg.progressPercent,
          isDelayed,
          delayDays: delayDays > 0 ? delayDays : undefined,
          isCriticalPath: seg.task.isCritical,
          assignees: seg.assignments.map(a => ({
            id: a.resource.id,
            name: a.resource.name,
          })),
        });
      }

      // 세그먼트 종료 이벤트
      if (seg.endDate >= windowStart && seg.endDate <= windowEnd) {
        events.push({
          date: seg.endDate.toISOString().split('T')[0],
          type: 'SEGMENT_END',
          taskId: seg.task.id,
          taskName: seg.task.name,
          segmentId: seg.id,
          segmentName: seg.name,
          isDelayed,
          delayDays: delayDays > 0 ? delayDays : undefined,
          isCriticalPath: seg.task.isCritical,
        });
      }

      // 진행 중인 세그먼트 (창 내에서 걸쳐있는 경우)
      if (seg.startDate <= windowStart && seg.endDate >= windowEnd) {
        events.push({
          date: centerDate.toISOString().split('T')[0],
          type: 'SEGMENT_ACTIVE',
          taskId: seg.task.id,
          taskName: seg.task.name,
          segmentId: seg.id,
          segmentName: seg.name,
          progressPercent: seg.progressPercent,
          isDelayed,
          isCriticalPath: seg.task.isCritical,
        });
      }
    }

    return events.sort((a, b) => a.date.localeCompare(b.date));
  }
}
```

```typescript
function usePresentationMode() {
  const [isPresentation, setIsPresentation] = useState(false);

  const enter = useCallback(async () => {
    await document.documentElement.requestFullscreen();
    document.documentElement.classList.add('presentation-mode');
    setIsPresentation(true);
  }, []);

  const exit = useCallback(() => {
    document.exitFullscreen();
    document.documentElement.classList.remove('presentation-mode');
    setIsPresentation(false);
  }, []);

  // ESC 키 감지
  useEffect(() => {
    const handler = () => {
      if (!document.fullscreenElement) setIsPresentation(false);
    };
    document.addEventListener('fullscreenchange', handler);
    return () => document.removeEventListener('fullscreenchange', handler);
  }, []);

  return { isPresentation, enter, exit };
}
```
