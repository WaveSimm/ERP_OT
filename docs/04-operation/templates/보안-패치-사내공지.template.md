**{배포일} {시간}**부터 ERP-OT 시스템에 보안 강화 업데이트가 적용됩니다.
업무 흐름에 영향 있는 변경 사항을 안내 드립니다.

## 사용자 행동 필요

### 1. 1회 강제 재로그인

업데이트 적용 시점에 **모든 사용자가 1회 자동 로그아웃**됩니다.
- 화면에 안내가 뜨면 다시 로그인하시면 됩니다.
- 기존 비밀번호는 그대로 유효합니다.

**원인**: 인증 토큰 저장 방식이 보안 강화로 인해 기존 토큰을 모두 무효화합니다.

### 2. 다른 기기 동시 로그인 가능

이전: 새 기기에서 로그인하면 기존 기기는 자동 로그아웃됐습니다.
변경 후: **회사 PC + 자택 PC + 모바일** 동시 로그인 가능.

**다른 기기 모두 로그아웃** 메뉴가 사용자 메뉴(우상단 프로필) 안에 추가됩니다.
보안 우려 시 (예: 회사 PC를 잠시 떠난 사이에 본인 토큰이 노출된 것 같다 등)
이 메뉴로 모든 기기 세션을 한 번에 종료할 수 있습니다.

### 3. 로그인 5회 실패 시 1분 잠김

브루트포스 공격 방어를 위해 **로그인 비밀번호 5회 연속 실패 시 1분 동안 잠김**.
- 비밀번호를 잊으신 경우 잠금 해제까지 기다리거나 운영팀에 문의 부탁드립니다.
- 정상 사용 환경에서는 영향 없습니다.

## 사용자 인지 못하는 변경

다음 사항은 사용자 화면 변화 없이 적용됩니다 (참고용):

- 인증 토큰을 브라우저 저장소가 아닌 보안 쿠키로 변경 (XSS 공격에 안전)
- 활성 토큰 시간을 8시간 → 1시간으로 단축 (자동 갱신은 평소처럼 동작)
- DB·내부 통신 채널 비밀번호 회전
- 보안 응답 헤더 5종 추가 (HSTS, X-Frame-Options, CSP 등)
- API 에러 응답 형식 표준화

## 영향 받는 화면 (회귀 가능성)

업데이트 직후 다음 화면이 정상 동작하는지 확인 부탁드립니다:

- 로그인 / 로그아웃
- 홈 화면 (내 작업, 알림, 게시판)
- 게시판 글 작성
- 작업비고 작성
- 결재 신청 / 처리
- 첨부파일 업로드 / 다운로드
- 검색

이상 발견 시 운영팀에게 즉시 알려 주시기 바랍니다.
