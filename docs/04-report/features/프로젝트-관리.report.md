## 6. 학습 및 회고 (Lessons Learned & Retrospective)

### 6.1 잘된 점 (Keep)

- **설계 문서 기반 구현**: v1.5 Plan + v2.0 Design의 명확한 구조 덕분에 구현 방향이 일관되어 초기 87%에서 93%로 개선 가능

- **분할된 아키텍처**: Clean Architecture 4-layer로 인해 기능별 추가(ResourceGroup, DashboardConfig)가 용이했음

- **다중 세그먼트 설계**: Task와 TaskSegment의 역할 분리로 인해 복잡한 자원 배정 시나리오 구현이 직관적

- **CPM 알고리즘 사전 검증**: Kahn + Forward/Backward Pass를 사전에 설계하여 의존성 체인 계산이 정확하고 성능 우수

- **반복형 PDCA**: Act 단계에서 1회 반복으로 87% → 96% 달성, 진정한 순환 개선 경험

- **테스트 우선성**: 82% 커버리지로 리팩토링 시 버그 조기 발견

### 6.2 개선할 점 (Problem)

- **UI/UX 미완성**: 템플릿 라이브러리 전용 페이지, 날짜 일괄 이동, 히스토리 타임라인 등 82% → 90%는 UI 추가 필요

- **비동기 처리 미흡**: 대용량 프로젝트 복제 시 동기 방식 사용 → Job Queue 미적용으로 응답 지연 가능성

- **외부 연동 부재**: Work Order 연동(FR-28), 파일 스토리지 백엔드 미분리 (로컬 볼륨 고정)

- **알림 Cron 누락**: 지연 위험 자동 감지 Cron은 Phase 9로 미연, Phase 실행 전까지 수동 확인 필요

- **환경 변수 관리**: 2건의 Warning 레벨 이슈 (JWT_SECRET, DB 연결 정보 하드코딩 위험)

- **초기 Scope 과대평가**: 48개 사용자 스토리 중 실제 v1.0은 44개 구현 → 초기 예측 오차

## 9. 핵심 성과

### 9.1 기술적 성과

✅ **Clean Architecture 4-layer 완성**
- API Layer: 10개 라우트 파일 (REST + JSON 기반)
- Application Layer: 8개 비즈니스 서비스
- Domain Layer: 핵심 엔티티 + CPM 알고리즘
- Infrastructure Layer: Prisma Repository + Redis Cache

✅ **CPM 알고리즘 구현**
- Kahn's Algorithm: 순환 의존성 감지
- Forward Pass: ES/EF 계산 (Early Start/Finish)
- Backward Pass: LS/LF 계산 (Late Start/Finish)
- Critical Path: Float = 0인 경로 자동 식별

✅ **다중 세그먼트 모델링**
- Task: 논리적 컨테이너 (effectiveStart/End 계산값)
- TaskSegment: 실행 단위 (독립 일정 + 자원)
- SegmentAssignment: 세그먼트별 % 또는 h/day 배정
- 자동 진행률 가중 평균 계산

✅ **그룹 계층 시스템**
- L1 (부서/고객사/프로그램) + L2 (세부 그룹)
- 2단계 제한 + DB 제약 강화
- 롤업 집계: 예산/진행률/이슈 자동 계산
- 중복 프로젝트 제거 로직 구현

✅ **템플릿 & 복사 시스템**
- 절대 날짜 → dayOffset 변환 (템플릿 저장)
- dayOffset → 절대 날짜 변환 (인스턴스화)
- 부분 가져오기: 선택 마일스톤/태스크만
- 의존성 보존 + 날짜 오프셋 적용

### 9.2 비즈니스 성과

✅ **프로젝트 관리 단일화**
- 스프레드시트 → 통합 플랫폼 이전
- 프로젝트 현황 조회: 30분+ → 실시간 (1분 이내)

✅ **자원 과부하 예방**
- 크로스-프로젝트 배정률 실시간 가시화
- 세그먼트 기간 기반 정확한 집계
- 100% 초과 시 자동 경고

✅ **일정 영향 자동 분석**
- 의존성 체인 변경 즉시 전파
- What-if 시뮬레이션 (저장 없이)
- 크리티컬 패스 자동 강조

✅ **Baseline을 통한 변경 추적**
- 다중 Baseline (최대 5개) 지원
- 세그먼트 단위 스냅샷
- 편차 자동 계산 + 간트 오버레이

✅ **협업 체계 구축**
- 태스크 댓글 + @멘션
- 파일 첨부 (설계도, 작업지시서)
- 활동 피드 (실시간 변경 이력)
- WebSocket 동기화
