## 6. Gaps Found

**Yellow (minor, non-blocking)**:

1. **MILESTONE_SINGLE_SEGMENT vs auto-correction**
   - Design §4.3 의사코드는 `endDate = startDate` 자동 보정 표현
   - 실제 구현: 두 번째 segment 시도 시 HTTP 409 반환 (더 방어적)
   - 동작은 더 안전하나 Design을 실제 API 계약에 맞춰 갱신 권장

2. **QA-3 시각 검증 명시 부족**
   - 마이그레이션 데이터 검증 통과 (sortOrder/parentId/segment 날짜)
   - 갠트 ◆ 시각 screenshot/curl은 별도로 첨부 안 됨
   - 데이터로부터 시각 PASS 추정 가능

3. **Plan §10 "dueDate as CPM constraint → discarded"** 항목
   - 이전 PDCA에서 애초 구현되지 않았으므로 코드 변경 없음 (Plan §1.3에서 명시)
   - 회귀 작업은 정확히 예방됨

**Red (blocking)**: 없음
