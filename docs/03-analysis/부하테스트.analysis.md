## Gap List

### G1. Round 4-6 production 모드 측정 실패 (CRITICAL — Scope Split 처리)

- Plan v1.0 §3.3 매트릭스 + 결정 #4 "dev + prod 둘 다 측정"이 핵심.
- 사유: Next.js prod 빌드가 type error + useSearchParams prerender 에러로 빌드 실패.
- **조치**: 별도 PDCA `prod-빌드-정리`로 분리. 본 PDCA Plan v1.1 / Design v1.1로 scope 재정의.
- Severity: High (원본 scope 기준), 0 (split 후)

### G2. NFR 정량 수치 일부 누락 (MEDIUM)

- Ollama throughput 건수/초 미측정 (CPU%만)
- DB connection 활성 수치 raw data 보존, 보고서 본문 통합 누락
- **조치**: 보고서 v1.1에 raw data 기반 표 추가 (Act 단계 처리)
- Severity: Low (raw data 있음, 통합만 누락)

### G3. Cleanup 미실행 (LOW)

- Design §6.3 "검증: cleanup 후 0건" 미수행
- 사유: 사용자 결정으로 분석 보존
- 스크립트 자체 dry-run 검증 완료
- **조치**: 분석 종료 후 1회 실행 + 보고서 부록에 검증 결과 추가
- Severity: Low (의도적 보류)

### G4-G6. Out-of-Scope 일치 (NOTE — deviation 아님)

- L11 댓글 작성자 필터 미적용 → Design §2.2 OoS 일치
