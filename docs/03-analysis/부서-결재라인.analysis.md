## 설계 초과 구현 (Bonus)

| 항목 | 설명 |
|------|------|
| soukwalUserId, daepyoUserId on Department | 3대표이사 체계 지원 (설계 당시 미고려) |
| thirdApproverId on ApprovalLine | 3단계 결재 (팀원→팀장→총괄이사→대표이사) |
| ApprovalLineService.bulkSetAll() | 전사 일괄 결재라인 설정 |
| GET /api/v1/approval-lines/me | 현재 사용자 결재자 조회 (이름 포함) |
| GET /api/v1/users/members | 결재자 선택용 사용자 목록 (인증 사용자 접근) |
| Leave/OT approverId 프론트 override | 신청 시 결재자 직접 지정 가능 |
| 결재자 드롭다운 UI (attendance page) | 휴가/OT 신청 폼에 결재자 선택 드롭다운 |
| getApprover() 3차까지 반환 | 설계는 2차까지였으나 3차 포함 |
