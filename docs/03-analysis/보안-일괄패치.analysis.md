### 성능 영향 (예상 — 실측 미수행)

Design 임계: BASELINE-2026-05-02 대비 p95 ±20%, error rate +1pt

- helmet 헤더 추가: 응답당 ~200B (negligible)
- @fastify/rate-limit (in-memory, Redis 미사용): IP→counter lookup (sub-ms)
